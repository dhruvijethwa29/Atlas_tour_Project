# 🌍 Atlas Tour Project

## 📌 About The Project

**Atlas Tour** is a tourism website project designed to provide users with an easy and interactive way to explore destinations, view tour details, and plan their trips.

The project focuses on creating a user-friendly interface where users can discover places, check travel information, and manage their tour experience.

---

## ✨ Features

- 🏝️ Explore different tourist destinations
- 🗺️ View tour packages and details
- 👤 User-friendly interface
- 📱 Responsive design
- 🔐 User authentication system
- 📩 Contact and booking functionality
- 🖼️ Destination image gallery

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Bootstrap
